Backup Help
What is backed up?

Backups include all database data: users, groups, expenses, settlements, and settings. Uploaded files (receipts) are NOT included in the backup.

Automatic Backups

Daily automatic backups run at the configured time. The system keeps the most recent backups and automatically deletes older ones.

Best Practices

Download backups regularly and store them in a safe location. Always create a backup before performing major changes.
